HOW IT STARTED

Late 2021, October precisely. I had this strong love for 'CODING'. asked a couple of friends who were into the it already. Most of them weren't ready to share the process. I can't really remember how but I joined a community of ladies that code also without any clue of what HTML and CSS was about.

Even though I had so many youtube videos at my disposal, it wasn't enough for me to learn properly. Then by December, I came across EHEALTH FREE WEB SKILLS FOR LADIES. How glorious! I decided to enroll. it was my first time applying for any online class. Lucky as I may be, after some series of test I got accepted

FIRST CLASS AND LEARNING

I got a mail sometime in early January about my acceptance into the academy, it was an exciting news. The mail came with a date for the first class which was held via zoom. First class was really understandable, the mentor broke things down like he was teaching some 5year old child.

We were asked tojoin the forum, create a github account and download Visual studio code. We had an assignment for the first module, I totally found it very easy and submitted.We had few more days to another class so I kept on coding whatever came to mind. Second day of class came, the real deal started.

A BREAKTHROUGH

During the class, I got confused. There was so much going on, so much explanation and I was lost. It all seemed too much to learn at once. I did some research via google to find some answers as well. I decided to start the course via elearning site, maybe I would understand it all from there. And yes I did! From each video watched and challenge I answered, I got to understand it all better.

Where ever I had issues, I would screenshot and ask my mentor. She has been really patient and understanding. When I saw some projects I was supposed to code, my heart skipped a bit. I really didn't know where and how but I have the best learning platform, Here i am coding my blog!

HOW IT IS GOING

So far it has been amazing, lovely, simple and enjoyable. In less than 6months I know so much basics on HTML and CSS. Ofcourse I would say hardwork and thus consistency brought me this far. Coding isn't as hard as people or my friends made it seem. With total Dedication anyone can be a pro at this. I can't wait to learn some hacks about web development, Play around with some codes and design whatever I imagine as well
